## Breve Descrição

Neste projeto, documentei a implementação da minha primeira stack com AWS CloudFormation. O repositório contém um template YAML para provisionar uma instância EC2 com Apache HTTP Server, além de anotações, passo a passo da execução, principais insights e cuidados importantes sobre segurança, validação e limpeza dos recursos criados.

Durante o desafio, pratiquei conceitos de Infrastructure as Code, estrutura de templates CloudFormation, uso de parâmetros, criação de recursos, outputs e acompanhamento dos eventos da stack.
